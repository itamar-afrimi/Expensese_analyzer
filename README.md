# מנתח הוצאות חכם מבוסס AI

מנתח הוצאות אישי חכם שמשתמש ב-OpenAI GPT כדי לסווג עסקאות בנקאיות ישראליות, לייצר תובנות על דפוסי הוצאות, ולאפשר שיחה אינטראקטיבית על הכספים שלך.

## תכונות

- **סיווג חכם** — AI מסווג עסקאות לקטגוריות אוטומטית (לא מבוסס מילות מפתח)
- **תמיכה בבנקים ישראליים** — לאומי, הפועלים, דיסקונט, מזרחי-טפחות
- **תובנות AI** — ניתוח חכם של דפוסי הוצאות עם המלצות לחיסכון
- **צ׳אט אינטראקטיבי** — שאל שאלות בעברית על ההוצאות שלך וקבל תשובות מפורטות
- **מעקב תקציב** — הגדר תקציב לכל קטגוריה וקבל התראות על חריגות
- **תרשימים ויזואליים** — תרשימי עוגה, עמודות ומגמות חודשיות
- **מטמון חכם** — תוצאות סיווג נשמרות כדי לחסוך בעלויות API

## התקנה

```bash
cd Expensese_analyzer
pip install -r requirements.txt
```

## הגדרת מפתח API

הגדר את מפתח ה-OpenAI API כמשתנה סביבה:

```bash
export OPENAI_API_KEY="sk-your-key-here"
```

או הוסף אותו ב-`config.yaml`:

```yaml
openai:
  api_key: "sk-your-key-here"
```

## שימוש

### ניתוח הוצאות

```bash
# ניתוח קבצי CSV ספציפיים
python main.py analyze --files march_2026.csv feb_2026.csv

# ניתוח כל הקבצים בתיקייה
python main.py analyze --dir ./exports/

# סיווג מחדש (התעלם מהמטמון)
python main.py analyze --files march.csv --recategorize
```

הפקודה `analyze` מבצעת:
1. זיהוי אוטומטי של פורמט הבנק
2. סיווג עסקאות עם AI (GPT-4o-mini)
3. הצגת סיכום הוצאות לפי קטגוריה ותקציב
4. יצירת תרשימים (נשמרים בתיקיית `output/`)
5. יצירת תובנות AI (GPT-4o)

### צ׳אט אינטראקטיבי

```bash
python main.py chat --files march.csv feb.csv
python main.py chat --dir ./exports/
```

דוגמאות לשאלות:
- "כמה הוצאתי על אוכל בינואר?"
- "מה ההוצאה הכי גדולה שלי החודש?"
- "תשווה לי את ההוצאות של ינואר מול פברואר"
- "איפה אני יכול לחסוך?"

הקלד `יציאה` לסיום הצ׳אט.

## הגדרות

ערוך את `config.yaml` להתאמה אישית:

### קטגוריות
ניתן להוסיף, להסיר או לשנות קטגוריות. ה-AI יסווג לפי הרשימה שתגדיר.

### תקציבים
הגדר תקציב חודשי בש"ח לכל קטגוריה:

```yaml
budgets:
  מזון וסופרמרקט: 3000
  מסעדות ובתי קפה: 1500
  דלק ותחבורה: 800
```

## מבנה הפרויקט

```
main.py              — נקודת כניסה CLI
config.yaml          — הגדרות, קטגוריות ותקציבים
models.py            — מודלים (Transaction, MonthlyReport)
store.py             — מטמון מקומי לסיווגים
visualizer.py        — יצירת תרשימים
parsers/             — פרסרים לבנקים ישראליים
  base.py            — ממשק בסיס
  leumi.py           — בנק לאומי
  hapoalim.py        — בנק הפועלים
  discount.py        — בנק דיסקונט
  mizrahi.py         — מזרחי-טפחות
  detector.py        — זיהוי אוטומטי
agent/               — רכיבי AI
  prompts.py         — תבניות prompt בעברית
  categorizer.py     — סיווג חכם (GPT-4o-mini)
  insights.py        — תובנות AI (GPT-4o)
  chat.py            — צ׳אט אינטראקטיבי
output/              — תרשימים שנוצרו
data/                — מטמון סיווגים
```
